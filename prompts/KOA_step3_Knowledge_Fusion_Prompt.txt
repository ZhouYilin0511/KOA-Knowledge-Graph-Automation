Special normalization (syntactic reconstruction required): Examples of typical cases: “膝关节活动受限以膝关节为甚” should be uniformly normalized as “膝关节活动受限明显”; “膝关节活动不利以下蹲为甚” should be uniformly normalized as “下蹲时膝关节活动不利明显”. Other similar expressions should be reconstructed following the same logic. (III) Knee joint swelling / enlargement / redness and swelling / swelling pain / swollen pain / distending pain: 1. For example: “行走时不慎摔伤后感膝关节肿痛较前加重” should be normalized as “行走时不慎摔伤后膝关节肿痛加重”; “膝关节肿胀不适” should be normalized as “膝关节肿胀”. 3. (V) Activity impairment (such as knee-joint squatting / rising from a squat / ascending and descending stairs): 1. Principle of strict distinction among core status terms: The three core status terms “受限”, “困难”, and “不能” must be strictly distinguished and must never be used interchangeably. Normalization must be based on the core status word in the original description: 受限 indicates a reduced range of motion; 困难 indicates that the action can still be performed, but only with difficulty or pain; 不能 indicates that the action cannot be performed at all. For example, if the original description contains “受限”, the normalized form must retain “受限”; if it contains “困难” or “难以”, it should be normalized as “困难”; if it contains “不能” or “无法”, it should be normalized as “不能”. Modifiers associated with the core status term in the original description (such as “稍”, “略微”, and “明显”) should be retained after normalization, but the core status term itself must be normalized to one of these three forms. For example, “稍困难” should remain “稍困难” after normalization, and the modifier must not be changed. 2. Anatomical-site supplementation: The prefix “膝关节” should be added. When the expression simply describes an obstacle to performing an action, forms such as “下蹲困难” and “难以下蹲” should be uniformly normalized as “膝关节下蹲困难”. Expressions such as “难以蹲起” and “不易下蹲起” should, according to the general rules, be normalized as “膝关节蹲起困难”. Expressions such as “膝关节下蹲受限” and “膝关节下蹲有受限感” should be uniformly normalized as “膝关节下蹲受限”. 3. Word-order correction and mandatory format unification (special rule for “困难” expressions): For expressions whose core status term is “困难”, regardless of whether the original description contains temporal or scenario words such as “时” or “后”, they must be forcibly normalized into the fixed format “膝关节 + action + 困难”, with all scenario prefixes removed (such as “蹲起时”, “下蹲时”, and “上下楼梯时”). For example: “蹲起时膝关节困难” → “膝关节蹲起困难” “下蹲时膝关节困难” → “膝关节下蹲困难” “上下楼梯时膝关节困难” → “膝关节上下楼梯困难” If the original description contains a degree modifier such as “稍” or “略微”, it should be retained in the format “膝关节 + action + 稍困难”. For example: “蹲起时膝关节稍困难” → “膝关节蹲起稍困难” “上下楼梯时膝关节稍困难” → “膝关节上下楼梯稍困难” This rule has the highest mandatory priority and applies to all “困难”-type expressions that simply describe difficulty in performing an action without a specific trend of change or an explicit time-related modifier. Only when a description involves a specific trend of change (such as “加重”, “缓解”, or “明显”) or an explicit temporal relation (such as “劳累后”, “受寒后”, or “休息后”) should the scenario prefix be retained, and the expression should then be normalized in the format “scenario + anatomical site + action + 困难 / 受限 / 不能 + change”. For example: “膝关节下蹲困难遇寒冷加重” should be normalized as “受寒后膝关节下蹲困难加重” “难以膝关节上下台阶” should be normalized as “膝关节上下台阶困难” When an obstacle to performing an action is caused by a core symptom such as pain, the expression should be uniformly reorganized into the format “anatomical site + core symptom + 难以 + action”, with redundant linking words removed. For example: “因膝关节疼痛难以上下台阶” “难以上下台阶时膝关节疼痛” “上下台阶时膝关节疼痛难以耐受” should all be uniformly normalized as “膝关节疼痛难以上下台阶”. 4. Removal of redundant colloquial terms: Remove irrelevant colloquial expressions such as "逐渐出现", "因", "情况", "仍", and non-core modifiers such as "部分", "感", as well as guiding words like "伴". For example, “逐渐出现膝关节下蹲困难” and “膝关节下蹲困难仍稍困难” should both be normalized to “膝关节下蹲困难”. Similarly, “因膝关节疼痛难以蹲起” should have "因" removed, resulting in “膝关节疼痛难以蹲起”. Other expressions such as “伴下肢放射痛” should be normalized to “下肢放射痛”, and simple descriptions of action impairment like “膝关节难以蹲起情况” or “膝关节仍难以蹲起” should be standardized to “膝关节蹲起困难”. Descriptions like “膝关节下蹲受限”, “膝关节下蹲活动受限”, and “膝关节下蹲有受限感” should be uniformly normalized as “膝关节下蹲受限”, and expressions like “膝关节入厕有受限感” should be standardized to “膝关节入厕受限”. 5. Special normalization: Expressions like “不能下蹲起”, “不能蹲起身”, and “无法下蹲起身” should be uniformly standardized as “膝关节不能蹲起” (if the action is difficult but possible, it should follow the general rule and be normalized as “XX困难”, such as “膝关节不易蹲起” or “膝关节难以蹲起”, both of which should be standardized as “膝关节蹲起困难”). Descriptions like “下蹲起”, “蹲起身”, and “下蹲起身” should be uniformly standardized as “蹲起” (since “蹲起” covers both actions of squatting and rising). There should be a strict distinction between “蹲起” and “下蹲”. Terms like “上下楼”, “上、下楼”, and “上楼下楼” should be uniformly standardized as “上下楼梯”, while “上楼/下楼” should be standardized as “上楼梯/下楼梯”. If the original sentence contains “时” or other wording explicitly indicating simultaneity or a conditional relation, it should be normalized as “劳累时 ...”. For fatigue-related scenarios, unless the original sentence explicitly contains the character “时”, it must always be normalized as “劳累后”. For example: “劳累膝关节疼痛加重” → “劳累后膝关节疼痛加重” “劳累出现膝关节疼痛” → “劳累后膝关节疼痛” “每遇劳累膝关节疼痛易反复” → “劳累后膝关节疼痛易反复发作” At the same time, colloquial introductory words such as “每因”, “因”, “每遇”, “遇”, “出现”, “发作”, and “致” should be deleted. Splitting must strictly generate exactly the same number of independent entries as the number of coordinated components. For example, “无口干口苦” must be forcibly split into the two independent entries “无口干” and “无口苦”; omission of either is strictly prohibited. The original composite entry must be deleted, and the two split entries must be inserted sequentially in its original position. Under no circumstances may fewer entries be generated or any split entry be omitted. Negative expressions such as “无明显”, “未诉”, and “暂无” should all be uniformly simplified into the structure “无 + core symptom”. Redundant modifiers such as “有”, “稍”, “稍有”, “偶有”, “口部”, and “口腔” must all be deleted. The specific splitting and normalization rules are as follows: ① Splitting of affirmative composite descriptions: Expressions such as “有口干口苦”, “口苦口干”, “稍口干口苦”, and “偶有口干口苦” must be split into the two independent symptom entities “口干” and “口苦”, and the corresponding independent triplets must be supplemented. ② Splitting of negative composite descriptions: Expressions such as “无明显口干口苦”, “暂无口苦口干”, and “无口干口苦等不适” must be split into the two independent symptom entities “无口干” and “无口苦”, and the corresponding independent triplets must be supplemented. ③ Splitting of mixed present/absent composite descriptions: Expressions such as “有口干无口苦”, “口干无口苦”, “口干不苦”, and “稍口干无口苦” must be split into the two independent symptom entities “口干” and “无口苦”, and the corresponding independent triplets must be supplemented. Expressions such as “无口干有口苦”, “无口干微口苦”, “口苦无口干”, and “无口干稍口苦” must be split into the two independent symptom entities “无口干” and “口苦”, and the corresponding independent triplets must be supplemented. ④ Normalization of single symptoms (non-composite): For single “口干” descriptions: all affirmative expressions such as “口部口干”, “稍口干”, “有口干”, “口干欲饮”, and “自觉口干” should all be uniformly normalized as “口干”. All negative expressions such as “无明显口干” and “口腔无口干” should be uniformly normalized as “无口干”. For single “口苦” descriptions: all affirmative expressions such as “偶有口苦”, “稍有口苦”, and “口部口苦” should all be uniformly normalized as “口苦”. All negative expressions such as “口腔无口苦” and “无明显口苦” should be uniformly normalized as “无口苦”. ⑤ Dynamic-change normalization: Expressions describing improvement of symptoms, such as “口干好转” and “口干改善”, should all be uniformly normalized as “[symptom]较前好转”, for example, “口干较前好转”.
④ Physical signs: I. Global general rules: All fusion of Physical signs must follow the general rules below: ① Core semantics first: Identify the core Physical signs described (such as “畸形”, “肿胀”, “压痛”, “凸起”, “凹陷”, “皮肤破溃”), and perform normalization around that core sign. ② Degree-word normalization: “轻度”, “轻微”, and “稍” must all be uniformly normalized as “轻度”, and must be placed before the core Physical signs term (for example, “膝关节轻度肿胀”, “膝关节后侧轻度压痛”, “膝关节髌下囊轻度压痛”). “Wei shen”, “明显”, “较甚”, and “显著” must all be uniformly normalized as “明显”. This is a highest-priority mandatory rule with no exceptions and must be checked item by item. In addition, the following mandatory positional rule must be applied: for a simple structure consisting only of a basic anatomical site (such as “膝关节”) plus a Physical signs term plus “明显”, with no further anatomical localization or acupoint-related modifier, “明显” must always be moved before the core Physical signs term, yielding the structure “site + 明显 + physical sign”. Typical required corrections include: “膝关节压痛明显” → “膝关节明显压痛” “膝关节肿胀明显” → “膝关节明显肿胀” However, when the description contains any specific anatomical localization, directional modifier, or acupoint-related modifier beyond the basic site, such as “内侧”, “外侧”, “前侧”, “后侧”, “髌下”, “关节间隙”, “髌周”, etc., this is considered a “detailed modified description with specific location”, and in such cases “明显” must remain fixed at the end of the entire phrase and must never be moved forward. For example, “膝关节内侧压痛明显” is correct and must not be changed to “膝关节内侧明显压痛”. Other correct examples include but are not limited to: “膝关节髌下囊压痛明显” “膝关节髌周压痛明显” “膝关节后侧压痛明显” “膝关节内侧关节间隙压痛明显” This rule is intended to strictly distinguish between descriptions with and without detailed modifiers, thereby ensuring the accuracy of medical terminology. ③ Normalization of negation and symbols: This rule applies to all negative Physical signs descriptions containing explicit negation words (such as “不”, “未见”, “未见明显”, “未及”, “无”) or negative symbols (such as “（-）”). The following mandatory reconstruction steps must be followed without exception: Step 1: Negation-word normalization. All negation words (such as “未见”, “未见明显”, etc.) must be uniformly normalized as “无”. Regardless of whether the original negation word is already “无” or another form, the following steps must still be carried out, and the original word order must not be directly retained. Step 2: Identify the core Physical signs and anatomical site. Based on the original description, identify the core Physical signs term (for example, “压痛”, “肿胀”, “畸形”, “皮肤破溃”, “皮温升高”), and preserve all modifiers describing its anatomical localization, nature, or extent (such as “髌下囊”, “内侧”, “皮肤”), together forming the complete core Physical signs phrase. Such modifiers must never be inserted between the anatomical site and “无”. At the same time, determine the anatomical site corresponding to that physical sign. If the original description explicitly states a site (such as “膝关节” or “下肢”), that site must be used directly. If no site is explicitly stated, then supplementation must follow this order of priority: First review all descriptions of the same Physical signs (same core term) already appearing in the current medical record document. If any such description contains an explicit anatomical site, that site must be preferentially used for supplementation. If no reference site exists anywhere in the document, then the default site “膝关节” must be supplemented. Note that anatomical descriptors already embedded in the core Physical signs phrase, such as “皮肤”, do not themselves count as the site and must remain inside the core sign phrase. Step 3: Mandatory reconstruction of the format (the only absolute format allowed). The description must be reconstructed strictly into the fixed format “anatomical site + 无 + core Physical signs phrase”. For example: “下肢感觉未见明显异常” → core physical sign “感觉异常”, site “下肢” → normalized as “下肢无感觉异常” “膝关节皮肤未见破溃” → core physical sign “皮肤破溃”, site “膝关节” → normalized as “膝关节无皮肤破溃” “膝关节无明显压痛” → core physical sign “压痛”, site “膝关节” → normalized as “膝关节无压痛” (with the degree word “明显” removed) Note: even if the original negation word is already “无”, this reconstruction step must still be carried out to ensure that the final form strictly follows “site + 无 + core Physical signs phrase”. For example, “膝关节皮肤无破溃” must be reconstructed as “膝关节无皮肤破溃”. Rules for converting examination-result symbols: The original conversion rules remain unchanged: “（+）” should be converted into the corresponding positive Physical signs expression (for example, “膝关节压痛（+）” → “膝关节压痛”) “（-）” should be uniformly converted into “无 + physical sign term” (for example, “膝关节畸形（-）” → “膝关节无畸形”) “（±）” should be uniformly normalized as “轻度 + physical sign term” (for example, “膝关节压痛（±）” → “膝关节轻度压痛”) If the content in parentheses provides supplementary explanation, it should be rewritten into an explicit Physical signs description (for example, “膝关节畸形（外翻畸形）” → “膝关节外翻畸形”; “膝关节畸形（肥大）” → “膝关节肥大样畸形”). ④ Rule for splitting negative coordinated composite Physical signs: All composite descriptions of the form “site + 无 / 无明显 / 未见明显 + coordinated physical signs” (such as “膝关节无凸起凹陷”, “膝关节无渗血渗液”, “膝关节无红肿发热”, “下肢无明显肿胀疼痛”, “下肢无肿胀疼痛”) must undergo a mandatory two-step process: first normalize the negation word, then split the coordinated physical signs. Step 1: Uniformly normalize “无明显”, “未见明显”, etc. as “无”. Step 2: Split the coordinated core Physical signs terms one by one into independent Physical signs triplets. Before splitting, all vague redundant location words such as “关节局部” and “局部” must be removed and replaced with the standard anatomical site “膝关节”. After splitting, each resulting entity must follow the fixed format “site + 无 + single physical sign”. Each split single Physical signs entity must fully inherit all relations, linked entities, and attribute values of the original composite Physical signs triplet. If the original composite Physical signs corresponds to multiple triplets, then each split single physical sign must also correspond to the same number and types of triplets, so that the split entities correspond one-to-one with the original relations, without any omission. Examples: “Physical signs: 膝关节无渗血渗液” → “Physical signs: 膝关节无渗血” and “Physical signs: 膝关节无渗液” “Physical signs: 膝关节无凸起凹陷” → “Physical signs: 膝关节无凸起” and “Physical signs: 膝关节无凹陷” “Physical signs: 下肢无明显肿胀疼痛” → “Physical signs: 下肢无肿胀” and “Physical signs: 下肢无疼痛” “Physical signs: 膝关节无红肿发热” → “Physical signs: 膝关节无红肿” and “Physical signs: 膝关节无发热” Special attention: “膝关节红肿热痛” is an orthopedic-specific composite Physical signs entity and does not need to be split. By contrast, expressions such as “红肿发热” and “凸起凹陷” are ordinary coordinated Physical signs and must still be split according to this rule. This rule is mandatory and has the highest priority for processing Physical signs entities. All eligible descriptions must be split without exception. ⑤ Eliminate redundant expressions: Remove colloquial redundancies, non-core modifiers, and repetitive expressions (for example, “膝关节内翻畸形改变” should be standardized as “膝关节内翻畸形”). Retain the core physical signs and key descriptive qualities (such as "骨性," "肥大样," "凹陷性"), as well as specific anatomical information (such as “膝关节髌上囊,” “膝关节内侧,” “膝关节关节间隙”). Standardize terminology (for instance, standardize “肤温” as "皮温" and “肤色” as "皮色"). ⑥ Anatomical site standardization: Other explicitly mentioned anatomical sites (such as “下肢,” “小腿,” “四肢”) must retain the original description and must not be arbitrarily changed to “膝关节.” If no anatomical site is explicitly mentioned, use the following priority order to supplement the anatomical site: Review all existing descriptions of the same physical sign (core term matching), whether affirmative or negative. If an anatomical site is specified in any of them, use that site for supplementation. If no site is found in the document, supplement with the default site “膝关节.” When a general anatomical term like “皮肤” is included, it is not considered the anatomical site and should remain within the physical sign. Directional terms like "双," "左," and "右" must be omitted, retaining only the physical sign description. Specifically, terms like “关节局部” and “膝部” must be standardized as "膝关节," and redundant modifiers like "局部" and "部" should be removed. II. Entity type correction rules (distinguishing between symptoms and physical signs): Before proceeding with entity standardization, Physical signs must be distinguished from Symptoms based on core semantics. Physical signs refer to abnormalities that can be objectively discovered or measured through physical examination (such as swelling, deformity, tenderness), while Symptoms refer to subjective sensations or functional abnormalities reported by the patient (such as pain, insomnia). The following descriptions are considered Symptoms and should be classified separately, without applying subsequent Physical signs fusion rules: Knee pain-related symptoms: All subjective descriptions with “膝关节疼痛” as the core, must be standardized as symptom entities, along with basic terminology normalization: “膝关节疼痛” “膝关节疼痛明显” (Originally “膝关节疼痛为甚” must be normalized to “膝关节疼痛明显” and corrected to symptom type) “膝关节腘窝后内侧疼痛明显” (Originally “膝关节疼痛，腘窝后内侧为甚” must be normalized to “膝关节腘窝后内侧疼痛明显” and corrected to symptom type) “下蹲/久坐/久站时膝关节疼痛加重” “膝关节屈伸活动时疼痛” (Originally “膝关节屈伸活动时膝关节疼痛” removes repetitive “膝关节”) “膝关节肿痛” (Originally “膝关节肿胀痛” standardized as this) “膝关节过屈痛” (Originally “过屈痛” supplemented with "膝关节" prefix). The 膝关节活动度 refers to objective measurements of the joint’s range of motion (such as “膝关节活动度正常/尚可/异常”), which is categorized as Physical signs. If a 膝关节活动度 description is misclassified as a symptom, it must be corrected to Physical signs category and standardized accordingly: “膝关节活动度可” → “膝关节活动度正常” “膝关节活动度尚可” → “膝关节活动度尚可” When “膝关节活动度” is mentioned, always ensure the correct categorization as Physical signs, and do not delete the original triplet entry, maintaining all relationships, attributes, and structure. Other descriptions that are not referenced here should follow subsequent Physical signs standardization rules. III. Expressions such as “膝关节关节间隙压痛明显”, “膝关节关节间隙明显压痛”, “膝关节间隙压痛明显”, and “膝关节间隙明显压痛” should all be uniformly normalized as “膝关节关节间隙压痛明显”. Acupoint tenderness: Expressions such as “血海压痛” and “膝关节血海穴压痛” should be uniformly normalized as “膝关节血海压痛”. Expressions such as “腧穴压痛” and “膝关节腧穴压痛” should be uniformly normalized as “腧穴压痛”. The negative form should be “腧穴无压痛”. Tenderness at multiple locations: Descriptions such as “膝关节多处压痛” and “膝关节多处穴位点压痛” should be uniformly normalized as “膝关节多处穴位点压痛”. Diffuse tenderness of the knee joint: Descriptions such as “膝关节广泛性压痛” should be uniformly normalized as “膝关节广泛压痛”. Note that “膝关节广泛压痛”, “膝关节周围压痛”, and “膝关节周围广泛压痛” are three distinct Physical signs entities and must each be retained in full. When the tenderness description includes a specific anatomical site such as the patella or joint space, for example “膝关节髌骨周围广泛压痛”, the full expression must be retained, and no anatomical site, directional modifier such as “周围”, or range/severity modifier such as “广泛” may be omitted. ⑦ Motion / range-of-motion-related Physical signs: Limitation of motion: Specific types of motion limitation (such as flexion-extension, rotation, and weight-bearing) must be retained exactly as originally described and must not be merged or confused. Only degree terms should be normalized (such as “轻度受限”, “明显受限”, and “无受限”). For example, “膝关节活动稍受限” should be normalized as “膝关节轻度活动受限”, and prefixes should be supplemented as needed (for example, “膝活动受限” → “膝关节活动受限”; “偶有交锁” → “膝关节偶有交锁”). For example, when “膝关节屈伸不利” appears as a Physical signs description, the original core term “不利” must be retained, and it must not be changed to “膝关节活动不利” or “膝关节屈伸受限”. Substitution with “受限” or any other synonym is strictly prohibited. Abnormal sensations: Clicking: “膝关节弹响” (general form): expressions such as “膝关节可闻及弹响声” and “膝关节可闻及弹响” should all be uniformly normalized as “膝关节弹响”. “膝关节活动时弹响” (specific scenario): expressions such as “膝关节活动异响” and “膝关节活动时有弹响” should all be uniformly normalized as “膝关节活动时弹响”. “膝关节屈伸活动时弹响明显” (with degree): expressions such as “膝关节屈伸活动弹响明显” must retain the degree description and be uniformly normalized as “膝关节屈伸活动时弹响明显”. Friction sensation: “膝关节摩擦感” (general form): expressions such as “膝关节可扪及摩擦感” and “骨擦音（+）” should all be uniformly normalized as “膝关节摩擦感”. “膝关节屈伸活动时有摩擦感” (specific scenario): expressions such as “膝关节屈伸时摩擦感” and “屈伸活动时有摩擦感” should all be uniformly normalized as “膝关节屈伸活动时有摩擦感”. Range-of-motion assessment: Knee joint range of motion: Descriptions related to knee-joint range of motion such as “膝关节活动度可” should be uniformly normalized as “膝关节活动度正常”. Descriptions such as “膝关节活动度尚可” should be uniformly normalized as “膝关节活动度尚可”. When normality is explicitly stated, the normalized form should be “膝关节活动度正常”. Abnormal descriptions related to knee-joint range of motion, such as “膝关节活动度欠佳”, “膝关节活动度（±）”, and “膝关节活动度明显受限”, should all be uniformly normalized as “膝关节活动度异常”. Descriptions that mention knee-joint range of motion but do not provide an explicit status, such as “膝关节活动度”, “膝关节活动/活动度相关描述”, and “膝关节因疼痛活动度无法查”, should all be uniformly normalized as “膝关节活动度无法查”. Patellar mobility: Patellar-mobility descriptions must be strictly normalized into one of the following standard levels: “膝关节髌骨活动度正常”, “膝关节髌骨活动度尚可”, “膝关节髌骨活动度欠佳”, and “膝关节髌骨活动度差”. The expression “可” corresponds exclusively to “正常”, and “可” and “尚可” must be strictly distinguished and must not be confused. Symbol standardization must be applied mandatorily and without exception, and all symbols must be removed completely: “膝关节髌骨活动度可”, “髌骨活动度可”, and “膝关节髌骨活动度（-）” → “膝关节髌骨活动度正常” “髌骨活动度I°” and “髌骨活动度尚可” → “膝关节髌骨活动度尚可” “髌骨活动度不佳”, “膝关节髌骨活动度（±）”, and “髌骨活动度（±）” → “膝关节髌骨活动度欠佳” “髌骨活动度差” and “髌骨活动度（+）” → “膝关节髌骨活动度差” During execution, for all descriptions whose core term is “髌骨活动度”, this grading-mapping rule must be applied directly. The rule in “negation and symbol normalization” that maps “（-） → 无 + physical sign term” must not be applied simultaneously, in order to avoid ambiguity or incorrect results. Gait and muscle strength: Descriptions related to “跛行” should all be uniformly normalized as “下肢跛行步态”. Descriptions related to “肌力正常” should all be uniformly normalized as “下肢肌力正常”. ⑧ Hyperextension / hyperflexion: Single-action format: When “过伸” and “过屈” are described separately in the original text as independent Physical signs, the prefix “膝关节” must be retained in full, and they must be independently normalized as: “膝关节过伸（+）”, “膝关节过伸（±）”, “膝关节过伸（-）” “膝关节过屈（+）”, “膝关节过屈（±）”, “膝关节过屈（-）” Omission of the “膝关节” prefix is strictly prohibited, and merging is also prohibited. For example: “过伸试验（+）”, “过伸（+）”, and “膝关节过伸（+）伴疼痛感” should all be uniformly normalized as “膝关节过伸（+）” “过屈（±）”, “过屈（+-）”, and “膝关节过屈试验（±）” should all be uniformly normalized as “膝关节过屈（±）” Combined format: Only when the original text explicitly treats the two as one combined test (such as “膝关节过屈过伸（+）”, “膝关节过屈过伸（±）”, or “膝关节过屈过伸（-）”) may they be normalized into a combined format. For example: “过屈过伸试验（+）”, “膝关节过屈过伸痛（+）”, and “膝关节过屈（+）过伸（±）” should all be uniformly normalized as “膝关节过屈过伸（+）” “膝关节过屈（±）过伸（±）” should be uniformly normalized as “膝关节过屈过伸（±）” Negative expressions should be uniformly normalized as “膝关节过屈过伸（-）”. ⑨ VAS score: Format unification: Descriptions such as “数字疼痛评分” should all be uniformly normalized as “VAS评分”. The format must be strictly standardized as “VAS评分X分”. The character “约” should be omitted, while decimal values should be retained. For example, “VAS评分约4.5分” should be normalized as “VAS评分4.5分”. Range expressions should be standardized as “VAS评分X-X分”. For example, “VAS大约4-5分” should be normalized as “VAS评分4-5分”. Irrelevant modifiers should be removed. For example, “膝关节疼痛严重时VAS评分7-8分” should be normalized as “VAS评分7-8分”. If the score is only described as positive without a specific numerical value, it should be uniformly normalized as “VAS评分（+）”. ⑩ Special test-related Physical signs: General rule: The names of special-test entities are themselves standard terms, and no anatomical prefix (such as “膝关节”) may be added. If the original description incorrectly includes an anatomical prefix (for example, “膝关节髌骨研磨试验”), that redundant prefix must be removed, and only the core test name should be retained. This is a highest-priority rule and overrides all other anatomical-site supplementation rules. Mandatory iron rule: Every special-test-related triplet must be traversed in full, one by one, without deleting, omitting, or skipping any original entry. Only normalization and correction may be performed; no test entry may be omitted without justification. Uniform result format: For each test, only the three standard original symbol formats “（+）”, “（±）”, and “（-）” may be retained, such as “浮髌试验（+）” and “抽屉试验（-）”. It is strictly prohibited to convert these original symbols into any other expressions such as “positive” or “negative”. Mandatory merging of equivalent tests: The traditional “回旋挤压试验” (also called “旋转挤压试验”) must be uniformly normalized as “McMurray test”. “磨髌试验” and related expressions must be uniformly normalized as “patellar grinding test”. Anterior and posterior drawer tests and related expressions must be uniformly normalized as “drawer test”. Expressions such as “膝关节内侧侧方应力试验”, “侧方试验”, “膝关节内、外侧应力试验”, and “内外翻应力试验” must all be uniformly normalized as “lateral stress test”. Important distinction: Grinding test and patellar grinding test are two different tests and must be normalized separately; they must never be confused. Among them, grinding test must be uniformly normalized into the following three forms only: “研磨试验（+）” “研磨试验（±）” “研磨试验（-）” For example: “膝关节研磨试验（+）”, “研磨试验(+)” → “研磨试验（+）” “膝关节研磨试验（±）”, “研磨试验(±)” → “研磨试验（±）” “膝关节研磨试验（-）” → “研磨试验（-）” “膝关节研磨试验不配合” should have the redundant “膝关节” prefix removed and be normalized as “研磨试验不配合”. Other test categories: “压髌征” should be normalized as “压髌试验” “4字征”, “‘4’字征”, and related variants should all be uniformly normalized as “4字征” “膝关节侧方挤压试验” should be normalized as “侧方挤压试验” ⑪ Tongue, coating, and pulse-related Physical signs: Splitting rule: Merged descriptions of tongue and coating, or tongue, coating, and pulse, should be split into independent single entities of tongue, coating, and pulse, with punctuation between them removed. For example, “舌暗紫，边有瘀斑，苔薄白” should be split into the two separate entities “舌暗紫边有瘀斑” and “苔薄白”, and the corresponding triplets (Physical signs: xxx, TCM syndrome differentiation, TCM syndrome: xxx) should be supplemented. Terminology normalization: Tongue manifestations: “舌质” should be uniformly changed to “舌”. For example, “舌质淡红” should be normalized as “舌淡红”. Coating manifestations: “舌苔” should be uniformly changed to “苔”. For example, “舌苔薄白” should be normalized as “苔薄白”. Since “少苔” is the standard expression, forms such as “苔少” should be uniformly normalized as “少苔”. Order of combined pulse descriptions: The professional ordering rule of TCM pulse diagnosis—position, rate, shape, force—must be strictly followed. That is, the fixed priority order is: position (e.g., 浮, 沉, 伏) > rate (e.g., 迟, 数, 疾) > shape (e.g., 弦, 滑, 细, 涩) > force (e.g., 实, 虚, 洪). This order is an absolute mandatory standard. Any pulse description must be reconstructed according to this priority order, and pulse order must never be reversed without basis. When a combined pulse description includes a position pulse (such as 浮 or 沉), that position pulse must be placed first. For example: “脉沉弦”, “脉浮数”, “脉沉迟”, “脉沉细”, and “脉浮滑” are the correct forms. Nonstandard forms with the position pulse placed later, such as “脉弦沉”, “脉数浮”, “脉滑浮”, and “脉迟沉”, must be forcibly corrected to “脉沉弦”, “脉浮数”, “脉浮滑”, and “脉沉迟”, respectively. When no position pulse is present in a combination, the conventional order among shape-related pulse terms must be enforced: 弦 pulse must always be placed first. For example, “脉涩弦” should be normalized as “脉弦涩”, “脉细弦” as “脉弦细”, and “脉滑弦” as “脉弦滑”. 细 pulse must be placed before 涩 and 滑. For example, “脉滑细” should be normalized as “脉细滑”, and “脉涩细” as “脉细涩”. This rule applies to all pulse descriptions without exception, so as to ensure that the terminology remains unique, standardized, and professionally appropriate.
⑤ Auxiliary examinations: The fusion rules for auxiliary examinations are divided into two main categories: imaging examinations and laboratory tests. The specific norms are as follows: ① Imaging examinations: Magnetic Resonance Imaging (MRI): Any description containing “膝关节” (knee joint), “磁共振” (magnetic resonance), or its abbreviations such as “MRI”, “MR”, or “核磁” (nuclear magnetic) alongside “平扫” (plain scan) (e.g., "膝关节MRI", "膝关节磁共振平扫", "膝MRI", "膝关节MR", "膝关节MRI平扫", "膝关节磁共振平扫+膝关节平扫", etc.) should be uniformly standardized as “膝关节磁共振平扫” (knee joint MRI plain scan). X-ray examination: Any X-ray examination related to the knee joint (e.g., "膝关节X线", "膝关节X片", "膝关节X线片", "膝关节正侧位片", "膝关节正侧位DR", etc.) should be uniformly standardized as “膝关节正侧位片” (knee joint anteroposterior and lateral X-ray). Lower limb full-length examination: Any X-ray examination assessing the full length of the lower limb (e.g., "下肢全长正位", "下肢全长片", "下肢全长X线", etc.) should be uniformly standardized as “下肢全长正位片” (full-length X-ray of the lower limb). CT examination: For routine CT scanning of the knee joint (e.g., "膝关节CT", "膝关节螺旋CT平扫", "膝CT", "左膝关节CT", etc.) should be uniformly standardized as “膝关节CT平扫” (knee joint CT plain scan). For 3D CT examinations of the knee joint (e.g., "膝关节螺旋CT三维成像", "膝关节三维CT", etc.) should be standardized as “膝关节CT三维成像” (knee joint 3D CT imaging). Ultrasound examination: Any ultrasound related to the knee joint (e.g., "膝关节彩超", "膝关节超声检查", "膝关节B超", "单侧膝关节超声", "膝关节肌骨彩超", etc.) should be standardized as “膝关节超声检查” (knee joint ultrasound). For ultrasound examinations targeting lower limb vessels (e.g., "下肢深部动静脉彩超", "下肢血管彩超", etc.) should be standardized as “下肢血管彩超” (lower limb vascular color ultrasound). Imaging examination of other parts: For MRI of non-joint specific parts of the lower limb (e.g., "下肢MRI", "下肢MR", etc.) should be standardized as “下肢磁共振平扫” (lower limb MRI plain scan). Combined examination: For descriptions of examinations involving multiple parts, typically use a unified format (e.g., "膝+腰椎核磁", "腰椎+膝关节MRI", "腰椎、膝关节磁共振平扫", etc.) and standardize as “膝关节+腰椎磁共振平扫” (knee joint + lumbar spine MRI plain scan). Imaging examination exceptions: For independent examinations that do not fit into the above categories (e.g., "髌骨轴位片") or examinations clearly targeting non-knee joint areas (e.g., "腰骶椎MR平扫"), retain the original expression without merging. ② Laboratory tests: Bone mineral density test: Terms like “骨密度测定” and “骨密度” should be uniformly standardized as “骨密度测定” (bone mineral density measurement). Erythrocyte sedimentation rate (ESR): Terms like “血沉”, “红细胞沉降率”, “膝关节血沉” should be standardized as “血沉” (erythrocyte sedimentation rate). C-reactive protein (CRP): Terms like “C反应蛋白”, “超敏C反应蛋白”, “CRP”, “C反应蛋白测定” should be standardized as “C反应蛋白” (C-reactive protein). Rheumatoid factor (RF): Terms like “类风湿因子”, “类风湿因子测定”, “膝关节类风湿因子” should be standardized as “类风湿因子” (rheumatoid factor). Anti-streptolysin O (ASO): Terms like “抗链球菌溶血素’O’”, “膝关节抗链球菌溶血素’O’” should be standardized as “抗链球菌溶血素“O”” (anti-streptolysin O).
⑥ Auxiliary Examination Results: 1. General Rules: All fusion and unification of auxiliary examination results follow the general norms below: ① Negative Descriptions Standardization (Highest Priority): All negative descriptions containing phrases like "未见明显" (no obvious), "无明显" (no obvious), "未见" (not seen), "未及" (not reached), etc., must first be unified and strictly standardized as “无” (none), and retained as part of the core examination result. For example: “附近骨质未见明显异常” (no obvious abnormality in adjacent bone) → “附近骨质无异常” (no abnormality in adjacent bone) “未见骨折线” (no fracture line seen) → “无骨折线” (no fracture line) “未见明显滑脱征象” (no obvious signs of subluxation) → “无滑脱征象” (no subluxation signs). ② Removal of Position Words and Abbreviation of Body Parts: Only remove directional terms such as "双" (both), "左", "右" (left, right), "单侧" (unilateral), "双侧" (bilateral), etc. However, anatomical positioning terms such as “内侧” (medial), “外侧” (lateral), “前侧” (anterior), “后侧” (posterior), "近端" (proximal), "远端" (distal), "上方" (superior), "下方" (inferior), and "拼接" (fusion) should not be deleted. If there is no corresponding anatomical location, it must not be arbitrarily added. For example: “双下肢不等长” (both legs of unequal length) → “下肢不等长” (unequal lower limbs) “下肢基本等长” (lower limbs basically equal in length) → “下肢基本等长” (lower limbs approximately equal in length) “下肢拼接下肢不等长” (fusion of unequal lower limbs) → “下肢拼接不等长” (fusion of unequal lower limbs). For descriptions involving specific anatomical positions, such as “胫骨平台下方” (below the tibial plateau), “下方” must be retained, and anatomical terms should be fully preserved. For shorthand descriptions like "膝前" (knee front), "膝后" (knee back), “膝内” (medial knee), "膝外" (lateral knee), etc., they should be standardized as “膝关节前侧”, “膝关节后侧”, “膝关节内侧”, and “膝关节外侧” (anterior, posterior, medial, lateral knee joint), respectively. These abbreviations are not acceptable. ③ Redundant Prefix Handling and Clear Regulations: Descriptions of injuries, tears, or swelling around the menisci, collateral ligaments, or soft tissues around the knee joint must retain the “膝关节” (knee joint) prefix. For example: “内侧副韧带周围损伤” (damage around the medial collateral ligament) → “膝关节内侧副韧带周围损伤” (damage around the medial collateral ligament of the knee). For anatomical locations that are already clearly specified, like “髌上囊” (suprapatellar pouch), “腘窝” (popliteal fossa), “髌骨” (patella), “股骨远端” (distal femur), and “胫骨平台” (tibial plateau), the redundant “膝关节” (knee joint) prefix should be deleted. For example: “髌上囊积液” (effusion in the suprapatellar pouch) → “髌上囊积液” (suprapatellar pouch effusion) “前交叉韧带损伤” (anterior cruciate ligament injury) → “前交叉韧带损伤” (anterior cruciate ligament injury). ④ Strict Terminology Distinction: Terms like “水肿” (edema), “肿胀” (swelling), etc., are independent terms and must not be interchanged. Descriptions like “水肿” (edema) must be preserved and cannot be replaced by “肿胀” (swelling). Additionally, terms like “囊性变” (cystic changes) should be standardized and degree modifiers (e.g., small, few, multiple) should be removed. For example: “小囊性变” (small cystic changes) → “囊变” (cystic changes). In describing long bone lesions (e.g., femur, tibia, fibula, humerus), terms like “上端” (upper end) should be standardized as “近端” (proximal), and “下端” (lower end) as “远端” (distal). For example: “股骨下端骨髓水肿” (bone marrow edema in the distal femur) → “股骨远端骨髓水肿” (bone marrow edema in the distal femur). ⑤ Complex Lesion Splitting Rules (Highest Priority, Preceding Other Connection Rules and Terminology Norms): For descriptions involving multiple lesions connected by words like “及” (and), “,” (comma), “、” (and), “并” (with), “伴” (accompanied by), “合并” (combined), etc., the rule is to split them into independent entities based on their anatomical structures or lesions, with each part inheriting the original relationships. For example: “内侧半月板后角及体部撕裂” (tear in the posterior horn and body of the medial meniscus) → “膝关节内侧半月板后角撕裂” (tear in the posterior horn of the medial meniscus) and “膝关节内侧半月板体部撕裂” (tear in the body of the medial meniscus). “前交叉韧带损伤伴内侧副韧带周围损伤” (anterior cruciate ligament injury with damage around the medial collateral ligament) → “前交叉韧带损伤” (anterior cruciate ligament injury) and “膝关节内侧副韧带周围损伤” (damage around the medial collateral ligament of the knee). For lesions connected by “伴” (with), “并” (and), etc., that describe the same mechanism of injury (e.g., “软骨损伤伴骨髓水肿” (cartilage injury with bone marrow edema)), these should not be split and should retain the connection. The connecting term “伴” (with) should be standardized as “并” (and). For example: “股骨远端软骨损伤并骨髓水肿” (cartilage injury of the distal femur with bone marrow edema) → “股骨远端软骨损伤并骨髓水肿” (cartilage injury of the distal femur and bone marrow edema). ⑥ Key Information Retention and Formatting: Three types of information must be preserved and formatted: Modifiers (Degree/Range): Words like “轻度” (mild), “轻微” (slight), “稍” (slightly) should be standardized as “轻度” (mild); terms like “重度” (severe), “严重” (serious) should be standardized as “重度” (severe). Uncertainty Modifiers: Terms like “可能” (possibly), “可疑” (suspected), “？” (?), “虑” (consider), “待排” (to be ruled out), etc., must be preserved in their original form and placed after the core entity. Grading Information: Should be standardized as Roman numerals with “级” (level) in parentheses, for example, “（III级）” (Level III), “（II-III级）” (Level II-III). ⑦ Core information must not be simplified: Disease names and injury-related modifiers in Laboratory and imaging findings (such as “变性”) must be retained in full and must never be arbitrarily deleted, so as to preserve diagnostic accuracy. In particular, any anatomical directional terms such as “内侧”, “外侧”, “前侧”, “后侧”, “近端”, “远端”, “上方”, and “下方” are all regarded as core information and must never be deleted or added without justification. ⑧ Removal of irrelevant information: Remove auxiliary expressions unrelated to the core examination result, such as “大致同前”, “请结合临床”, and “仍”, and retain only the core lesion information. For special descriptions enclosed in quotation marks (such as changes “股骨远端骨折内固定术后”), the original quotation marks and their positions must be preserved completely and must not be deleted or altered. This rule does not conflict with the above rule on mandatory normalization of negative descriptions; the negation word “无” must be retained as part of the core information. ⑨ Norm for combined descriptions: If a description contains one main entity combined with other manifestations (such as “A并B”), the fused result must fully list all combined entities. II. Category-specific fusion rules and application standards: The following category-specific fusion rules are applied on top of the global general rules. ⑨ Chinese Patent Medicines: The fusion and standardization rules are as follows: 1. Purification of Core Names: Remove all additional information in parentheses and retain only the generic drug name. For example, “藤黄健骨” should be standardized as “藤黄健骨片,” “瘀血痹” as “瘀血痹片,” and “盘龙七” as “盘龙七片.” Medicines like “仙灵骨葆” and “活络消痛” explicitly correspond to capsules, so they should be standardized as “仙灵骨葆胶囊” and “活络消痛胶囊.” External Preparations Standardization: For external medicines with vague dosage forms or those ending in “外用,” the dosage form should be added based on the generic name and common clinical usage, such as “软膏,” “贴膏,” “酊剂,” “气雾剂,” or “注射液.” For example, “三黄肿痛外用” should be standardized as “三黄肿痛软膏,” and “黄栀理伤膏” as “黄栀理伤贴膏.” 3. Remove Irrelevant Information: Remove non-core additional information in the medicine name and retain only the generic name of the drug. For example, “膝关节止痛贴” should be standardized as “止痛贴,” and “三黄膏贴敷” should be standardized as “三黄膏.” 4. Correction of Aliases, Abbreviations, and Misspellings: Standardize the name of Chinese patent medicines, correcting non-standard words and expressions to ensure accurate and unique names. For instance, product names like “伤速康” and “伤速康贴膏” should be standardized to the generic name “黄栀理伤贴膏.” Abbreviated forms should also be expanded to full standard names, e.g., “罗浮山风湿膏” should be standardized as “罗浮山风湿膏药.” Common errors in names, such as “藤黄建骨片” should be corrected to “藤黄健骨片,” and “元七骨痛町” should be corrected to “元七骨痛酊.” 5. Strict Category Correction: Based on the drug’s inherent properties (chemical composition and approval category), misclassified Chinese patent medicine entities should be corrected and moved to their appropriate categories. For example, entities like “中药封包” should be corrected to “中药处方,” and if the category is accurate and the name is a standard generic name, the category label must remain unchanged and should not be adjusted to another category. Any form of incorrect category adjustments is prohibited.
⑩ Chinese herbal medicine: The fusion of Chinese herbal medicine entities is based on unifying synonymous expressions and standardizing names toward commonly used standard herb names. All rules below are of the highest priority and must be enforced strictly: First, the specific fusion operations include the following: 1. Correct writing errors: Correct obvious typographical errors, redundant spaces, and nonstandard characters. For example, “红 花” should be normalized as “红花”, “三匕” as “三七”, and “沙菀子” as “沙苑子”. 2. For example: “当归（复方）” → “当归” “附片（黑顺片）” → “附片” “炒芥子[精]” → “芥子” “后下盐益智仁” → “盐益智仁” “干鱼腥草” → “鱼腥草” 3. Omit processing methods: For herb names that clearly belong to the same botanical or medicinal origin and differ only because processing changes their medicinal properties (such as 盐, 酒, 醋, 炙, 炒, 煅, 制, 烫, 蜜, 燀, 炮, 麸炒, etc.), all processing-related prefixes or suffixes must be removed unconditionally, and the herb must be merged into its base herb name. This rule has priority over all others. Unless a case is explicitly listed below under the prohibition of improper merging as one that must be preserved, it must be forcibly normalized. Recognition of processing methods includes but is not limited to the methods listed above. As long as the name contains words indicating processing and can be merged into the base herb, the processing marker must be removed. Standardize base herb names (mandatory): Only for structures of the form “生 + herb name”, such as “生甘遂” and “生甘草”, remove the prefix “生” and normalize to the base herb name. However, “生地黄” and “熟地黄” are fixed standard herb names and must not be modified. For example: “生甘遂” → “甘遂” “生地黄” and “熟地黄” remain unchanged. 5. Forcibly delete all suffixes indicating physical form or modern preparation: Suffixes indicating dosage form or physical form, including but not limited to “粉”, “片”, “超微饮片”, “颗粒”, “饮片”, and “粗粉”, must be removed unconditionally, retaining only the core herb name. For example: “三七粉” → “三七” “肉桂粉” → “肉桂” “甘草片” → “甘草” “黄芪颗粒” → “黄芪” 6. Forcibly complete abbreviations and normalize aliases: Completion of abbreviations: restore incomplete names to their full standard names. For example, “生地” should be normalized as “生地黄”, and “浙贝” as “浙贝母”. Forced unification of aliases: map aliases, variant names, and local names to standard common names. For example: “云木香” → “木香” “玄胡” and “元胡” → “延胡索” “白三七” → “竹节参” “公丁香” → “丁香” 7. Ignore dosage-form descriptions: Remove suffixes indicating physical form or modern preparation (such as “粉”, “片”, and “超微饮片”). For example: “肉桂粉” → “肉桂” “甘草片” → “甘草” Second, pay special attention to the following key principles and prohibitions: 1. Apply composite rules in sequence: When the same name involves multiple rules, they must be applied in the order 1 → 6 above. When a name contains both parentheses and a processing prefix, the order must be followed strictly: first apply Step 2 to remove the parentheses and their content, and then apply Step 3 to omit the processing marker, so that the processing prefix can be correctly identified and removed. For example: “炒芥子[精]” should first become “炒芥子” after removal of the bracketed content, and then “芥子” after removal of the processing marker “炒”. “酒萸肉(酒山萸肉)” should first become “酒萸肉” after removal of the parentheses, and then “山萸肉” after omission of the processing marker. 2. Improper merging is strictly prohibited: The following cases must never be merged and must remain as independent entities: Different medicinal parts: for example, “当归尾” (branch root) versus “当归” (main root), and “杜仲叶” (leaf) versus “杜仲” (bark). Conservative principle: preserve if uncertain: If it cannot be clearly determined, based on explicit evidence, which specific standard name the herb should be merged into (for example, when only “地黄” appears and it is impossible to determine whether it means 生地黄, 熟地黄, or 干地黄; or when only “牛膝” appears and it is impossible to determine whether it means 川牛膝 or 怀牛膝), the original description must be preserved and must not be forcibly modified. Forced fusion is strictly prohibited.

⑬ Western non-pharmacological therapy: The core principle of fusion and standardization is that normalization should be based on the core action/mechanism of the therapy, while the necessity of retaining anatomical-site information must be judged strictly. The specific rules are as follows: 1. Purification of the core therapy name: 1) Remove parentheses and their contents: Delete all parentheses and their contents from the therapy name (such as body part or parameter information). For example: “踝泵训练（主动+被动）” → “踝泵训练” “关节松动训练（膝关节）” → “关节松动训练” 2) Remove redundant action/status modifiers: Delete non-core modifiers such as “进行”, “加强”, “适当”, “逐步”, “在医师指导下”, “视情况”, “局部”, and “患肢”. For example: “局部冰敷” → “冰敷” “进行不负重功能锻炼” → “不负重功能锻炼” “视情况下地活动锻炼” → “下地活动锻炼” (“下地” is a core scenario qualifier and must not be deleted) “床上功能锻炼” → “功能锻炼” (“床上” is a removable scenario modifier; here, “功能锻炼” is the core term) 3) Handling of core site information: Case A: Remove redundant site words. When the therapy name itself already implies a specific site, or when the original site word is generic, it should be deleted. Examples: “加强下肢踝泵训练” → “踝泵训练” (“踝泵” already specifies the site) “患肢直腿抬高训练” → “直腿抬高训练” “四肢关节活动训练” → “关节活动训练” (“四肢” is a conventional default scope and may be deleted) Case B: Retain core site words. When the therapy name is a generic term and the site word is needed to specify the treatment target, the site word must be retained. Examples: “膝关节伸直训练” must be retained as “膝关节伸直训练”, and must not be simplified as “伸直训练” “股四头肌肌力训练” already contains the site information and should be retained as “股四头肌肌力训练” “腰背部关节松动训练” should be retained as “腰背部关节松动训练” 2. Merging synonymous therapies (normalization by core action/mechanism): After applying the above site-handling principle and purification rules, therapies with different wording but exactly the same core action/mechanism should be merged into one standard main name. ⑯ Surgery: Standardization Guidelines 1. Purification of Core Terminology Remove supplementary information in parentheses (such as alternate procedures, material descriptions, notes, etc.) and redundant modifiers (like "术中" or other non-core qualifiers), keeping only the core surgical procedure. The expressions inside parentheses are generally considered part of the core procedure and should be integrated into the procedure name, unless specified otherwise. For example: "缝合" and "(备缝合)" refer to completely different surgical concepts. Do not integrate "(备缝合)" into "缝合"; the original parentheses and the word "备" must be kept as is. All individual procedural units connected with "+" are considered core components of the surgery. Do not delete or merge any unit connected by "+", such as: "半月板部分切除修整成形（备缝合）" → “半月板部分切除成形（备缝合）” "关节清理（含游离体取出）" → “关节清理+游离体取出” "人工膝关节（单髁）表面置换术" → “人工膝关节单髁表面置换术” "人工全膝关节表面置换+髌骨成形术+术中C臂透视术" → “人工全膝关节表面置换+髌骨成形+C臂透视术” (Remove redundant "术中" and apply the rule of suffix "术" for compound procedures). 2. Single Procedure Joint Localization Standardization This rule applies only to single procedures (excluding compound surgeries connected by "+"). When the core site described in the procedure is "关节" (without specifying the joint type), it should be unified to “膝关节”. For example: "关节镜手术" → “膝关节镜手术” (according to this rule), and later simplified to “膝关节镜术”. "关节清理" → “膝关节清理”, later simplified to “膝关节清理术”. "关节抽吸术" → “膝关节抽吸术” (the word “术” is already included, no need to add another). 3. Removal of Non-Core Site Modifiers Remove non-essential anatomical site modifiers, such as "左", "右", "左侧", "右侧", etc. For example: "左人工全膝关节置换+髌骨成形术" → “人工全膝关节置换+髌骨成形术” 4. Uniformity of the "术" Suffix and Position Ensure all surgical names end with the character "术", as this is a highest-priority rigid rule. Regardless of whether the last unit in a compound surgery is the core procedure or an auxiliary operation (like C-arm imaging, joint cleaning, radiofrequency hemostasis, etc.), the suffix "术" must be retained at the end of the entire procedure description. The suffix "术" for auxiliary operations must also not be omitted. Specific rules: ① Single Surgery: If the original description ends with "手术", simplify it to "术". If it does not contain "术", then add it. For example: "膝关节镜手术" → “膝关节镜术” "关节镜手术" → “关节镜术” "关节清理" → “关节清理术” ② Compound Surgery (multiple operations connected by "+") You must delete the "术" suffix from all intermediate units except the last one, ensuring that the last operation in the compound description retains the "术" suffix. For example: "人工全膝关节表面置换术+髌骨成形术" → “人工全膝关节表面置换+髌骨成形术” "膝关节镜检查+半月板切除成形术+滑膜切除术" → “膝关节镜检查+半月板切除成形+滑膜切除术” "人工全膝关节表面置换+髌骨成形+C臂透视术" → “人工全膝关节表面置换+髌骨成形+C臂透视术” (Do not omit the "术" at the end). 5. Connecting Symbols and Compound Procedure Standardization If the surgical description contains the "+" connecting symbol, the connected parts are core procedural steps. The "+" symbol should be preserved.
